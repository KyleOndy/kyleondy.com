tmux

Quick notes on common usages
Updated: December 9, 2018

My leader key combo in tmux isctrl+space.

Copy text in tmux

  1. ctrl+space,[ to enter copy mode
  2. Move to text you would like copied
  3. ctrl+space to start highlighting. Selected text highlights to show selection.
  4. Move to end of text.
  5. enter copies text into clipboard.
  6. Do whatever
  7. ctrl+space,] to paste contents.

Handy shortcuts

  • ctrl+space,x to kill window
  • ctrl+space,X to kill window
  • ctrl+space,space to iterate through window layouts
