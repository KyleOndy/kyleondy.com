Debian

man debian
Updated: October 23, 2018

Upgrading Debian from one release to another.

sed -i 's/stretch/buster/g' /etc/apt/sources.list

Once /etc/apt/sources.list/etc/apt/sources.list is updated, update the running system.

apt update
apt upgrade
apt dist-upgrade
